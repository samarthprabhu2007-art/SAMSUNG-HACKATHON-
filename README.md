# ⚡ Answer to Unlock

## 📌 Overview

Answer to Unlock is a productivity web app that connects **studying with rewards**.

Instead of just tracking time, users must **prove they learned something** by answering questions before unlocking rewards like break time or social media usage.

---

## 🚀 Features

* 🔐 Login system (Name + Email)
* 🏠 Dashboard with 4 core sections:

  * Start Session
  * Quiz
  * Rewards
  * Progress
* 🎯 Study session setup
* 🧠 Quiz-based verification (core idea)
* 🎁 Reward unlocking system
* 👤 Profile editing (Name + Email)

---

## 🧠 Concept Flow

Start Session → Quiz → Reward → Progress

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript
* LocalStorage (for storing user data)
* CSS (inline styling)

---

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/YOUR-USERNAME/answer-to-unlock.git
```

### 2. Go to project folder

```
cd answer-to-unlock
```

### 3. Install dependencies

```
npm install
```

### 4. Start development server

```
npm run dev
```

### 5. Open in browser

```
http://localhost:5173
```
